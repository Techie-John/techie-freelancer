6. Chrome Extension Permissions

6.1 Required Permissions and Justification

  • activeTab:
    • Purpose: Access current profile page for analysis
    • Data accessed: Visible profile content only
    • When: Only when you click "Analyze" button
  • storage:
    • Purpose: Save preferences and login state
    • Data stored: Auth token, user settings
    • Location: Local browser storage
  • notifications:
    • Purpose: Notify when analysis is complete
    • Data: Analysis status messages
    • Control: Can be disabled in browser settings
  • tabs:
    • Purpose: Open results in dashboard
    • Data accessed: Current tab URL
    • Usage: Navigate to results page
  • downloads:
    • Purpose: Save profile screenshots locally
    • Data: Screenshots you initiated
    • Control: Saved to your Downloads folder
  • Host Permissions (upwork.com, linkedin.com):
    • Purpose: Analyze profiles on these sites
    • Data accessed: Public profile information
    • Limitation: Only when actively using extension
